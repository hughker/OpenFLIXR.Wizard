Welcome to your OpenFLIXR

Finishing this setup will automatically create random API keys and install Plex Media Server.

Before you can use OpenFLIXR, you need to mount ‘downloads’ (subdirectories will be created at boot time when mounted to a network share), ‘movies’, ‘series’, ’music’ and ‘comics’ folders from a location with enough free disk space like a NAS, external harddrive, etc.

To make things easier you can use Webmin to accomplish this. OpenFLIXR is case sensitive.

To be able to use OpenFLIXR with default settings, you must use the following directory structure!

If you need help, use the forums.
